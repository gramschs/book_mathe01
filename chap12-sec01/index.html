12.1 Stammfunktion

In diesem Abschnitt führen wir zunächst Stammfunktionen ein.

Lernziele¶

Lernziele

  • Sie wissen, was eine Stammfunktion ist.

  • Sie wissen, dass wenn es eine Stammfunktion gibt, es gleich unendlich viele Stammfunktionen gibt, die sich durch eine Integrationskonstante unterscheiden.

Stammfunktion¶

Was ist ... eine Stammfunktion?

Angenommen, die Funktion fff ist stetig. Dann wird die Funktion FFF Stammfunktion von fff genannt, wenn ihre Ableitungsfunktion gleich fff ist, also wenn gilt F′(x)=f(x)F'(x) = f(x)F′(x)=f(x).

Beispiel: Wir betrachten die Funktion f(x)=xf(x)=xf(x)=x. Mit ein bisschen Probieren fällt auf, dass die Funktion F(x)=12x2F(x)=\frac{1}{2}x^2F(x)=21​x2 abgeleitet genau xxx ergibt. Also ist die Funktion FFF eine Stammfunktion von fff. Jetzt probieren wir die Funktion F~(x)=12x2+1\tilde{F}(x)=\frac{1}{2}x^2+1F~(x)=21​x2+1 aus. Wenn wir diese Funktion ableiten, kommt

F~′(x)=(12x2+1)′=x+0=x\tilde{F}'(x) = \left( \frac{1}{2}x^2 + 1 \right)'= x + 0 = xF~′(x)=(21​x2+1)′=x+0=x

heraus. Das ist aber auch gleich der Funktion fff. Welches ist denn nun die richtige Stammfunktion?

Beide Funktionen sind eine Stammfunktion. Wenn eine Funktion fff eine Stammfunktion hat, dann hat sie gleich unendlich viele Stammfunktionen, denn es muss nur eine reelle Zahl hinzuaddiert werden und schon hat man eine neue Stammfunktion.

Wie viele Stammfunktionen gibt es?

Wenn eine Funktion fff eine Stammfunktion hat, hat sie gleich unendlich viele Stammfunktionen. Die Stammfunktionen unterscheiden sich nur dadurch, dass eine reelle Zahl am Ende dazuaddiert wird. Diese Zahl wird Integrationskonstante genannt.

Begründung: Wenn FFF die erste gefundene Stammfunktion ist, dann muss ihre 1. Ableitung gleich fff sein, also F′(x)=f(x)F'(x) = f(x)F′(x)=f(x) gelten. Dann basteln wir eine neue Funktion F~(x)=F(x)+c\tilde{F}(x) = F(x) + cF~(x)=F(x)+c mit c∈Rc\in\mathbb{R}c∈R. Das muss dann aber auch eine Stammfunktion sein, denn

F~′(x)=(F(x)+c)′=F′(x)+0=f(x).\tilde{F}'(x) = \left(F(x)+c\right)' = F'(x) + 0 = f(x).F~′(x)=(F(x)+c)′=F′(x)+0=f(x).

Wichtige Stammfunktionen¶

Zu Funktionen, die häufig im Maschinenbau vorkommen, sollten Sie die Stammfunktionen auswendig kennen.

  • f(x)=a⇒F(x)=ax+cf(x) = a \Rightarrow F(x) = ax + cf(x)=a⇒F(x)=ax+c

  • f(x)=xn⇒F(x)=1n+1xn+1+cf(x) = x^{n} \Rightarrow F(x) = \frac{1}{n+1} x^{n+1} + cf(x)=xn⇒F(x)=n+11​xn+1+c (nnn muss ungleich -1 sein)

  • f(x)=x−1=1x⇒F(x)=∣ln⁡(x)∣+cf(x) = x^{-1}=\frac{1}{x} \Rightarrow F(x) = |\ln(x)| + cf(x)=x−1=x1​⇒F(x)=∣ln(x)∣+c

  • f(x)=ex⇒F(x)=ex+cf(x) = e^{x} \Rightarrow F(x)=e^{x}+cf(x)=ex⇒F(x)=ex+c

  • f(x)=sin⁡(x)⇒F(x)=−cos⁡(x)+cf(x) = \sin(x) \Rightarrow F(x)=-\cos(x)+cf(x)=sin(x)⇒F(x)=−cos(x)+c

  • f(x)=cos⁡(x)⇒F(x)=sin⁡(x)+cf(x) = \cos(x) \Rightarrow F(x)=\sin(x)+cf(x)=cos(x)⇒F(x)=sin(x)+c

  • f(x)=11+x2⇒F(x)=arctan⁡(x)+cf(x)=\frac{1}{1+x^2} \Rightarrow F(x) = \arctan(x) + cf(x)=1+x21​⇒F(x)=arctan(x)+c
